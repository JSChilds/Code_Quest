INSTRUCTIONS

CodeQuest is a text-based adventure game. You will have to use your cunning, comprehension of coding and web development and basic reading comprehension to navigate and escape a treacherous dungeon.

There are four set commands you can type into the game:

GO: Allows you to move to another area. Type 'GO NORTH', 'GO WEST', 'GO EAST', or 'GO SOUTH' to move around the dungeon.

INSPECT: Allows you to inspect things in a room that have been specified in BOLD CAPITAL LETTERS. Type 'INSPECT ' followed by the object you wish to inspect.

TAKE: Allows you to pick up certain objects that have been specified to use them later on. Type 'TAKE ' followed by the object you want to take

USE: Allows you to use objects you've already picked up on other objects that have been specified. Type 'USE ' followed by the object you wish to use.

You will also have to solve riddles and answer questions by typing your answer, or your choice from a selection of answers.
